Basic visualization with angular classes:

Angular automatically attaches classes to input elements and forms. 
    .ng-pristine { border:1px solid Gold; }
    .ng-dirty.ng-valid { color:green; }
    .ng-dirty.ng-invalid { color:red; }
Again:
Property Class Description
$valid ng-valid Boolean true if valid, otherwise false.
$invalid ng-invalid Boolean true if invalid, otherwise false.
$pristine ng-pristine Boolean true if the form/input has not changed.
$dirty ng-dirty Boolean true if the form/input has changed.
custom ng-invalid-custom ngModelCtrl.$setValidity('custom', false);
