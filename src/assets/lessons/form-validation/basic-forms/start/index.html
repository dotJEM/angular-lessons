dotJEM Angular Lessons - Basic Forms
  • Create a user form - following should be the case:
    user name:
    required, minimum 3 chars and maximum 16 chars, only allow a-z, numbers, '_' and '-'.
    name:
    required, numbers and special characters are not allowed.
    date of birth:
    optional, but must be a valid date.
    email:
    required, should be a valid email.
